## 📋 Visão Geral

**BaaS Portal dos Bancos** é uma plataforma de gerenciamento de chaves PIX, DICT e clientes para instituições financeiras participantes do sistema de pagamentos instantâneos brasileiro, com conformidade total a LGPD e padrões internacionais de segurança.

### Características Principais

- ✅ **Gerenciamento de Chaves PIX** - Registrar, alterar e excluir chaves PIX
- ✅ **Consulta DICT** - Integração com dicionário de chaves do Banco Central
- ✅ **Gerenciamento de Clientes** - CRUD completo com validação de dados
- ✅ **Autenticação Segura** - Keycloak OAuth2 + OpenID Connect + MFA
- ✅ **Conformidade LGPD** - Criptografia, auditoria e direitos do titular
- ✅ **Dashboard Administrativo** - Relatórios em tempo real
- ✅ **APIs RESTful** - Documentação via OpenAPI/Swagger
- ✅ **CI/CD** - GitHub Actions com testes e segurança
- ✅ **Teste de Penetração** - Análise de vulnerabilidades contínua
